# 帛書德道經

## 什麼是帛書？

1973年，湖南省長沙市**馬王堆三號漢墓**出土了兩件珍貴的絲帛書寫老子，稱為：

- **甲本**（帛書老子甲）：抄寫年代較早，約西漢初期，字跡較工整，保留了更多古字形
- **乙本**（帛書老子乙）：抄寫年代稍晚，漢代隸書，因避漢惠帝劉盈諱而有部分字形改動

這兩本帛書距今已有**二千多年**的歷史，是目前發現最早的完整老子版本。

## 與流傳版的最大差異

### 1. 順序顛倒：德在前，道在後

| 流傳版（王弼本） | 帛書版 |
|---------|-------|
| 道經（第1-37章）在前 | 德經（相當於流傳第38-81章）在前 |
| 德經（第38-81章）在後 | 道經（相當於流傳第1-37章）在後 |

帛書版因此被稱為《**德道經**》，而非《道德經》。

### 2. 重要字詞差異

| 流傳版 | 帛書版 | 說明 |
|--------|--------|------|
| 常 | 恆 | 帛書用「恆」，傳世版避漢文帝劉恆諱改為「常」 |
| 谓 | 胃 | 帛書古字 |
| 又玄 | 有玄 | 第一章「玄之又玄」，帛書作「玄之有玄」 |

### 3. 文字更接近上古原貌

帛書保留了許多在流傳過程中被改動、簡化或誤抄的字詞，對研究老子原意具有重要學術價值。

## 閱讀說明

本站帛書德道經以**德經在前**的原始順序呈現：

- **德經**（帛書上篇）：對應流傳版第38-81章
- **道經**（帛書下篇）：對應流傳版第1-37章

每章均標注**對應流傳版章節**，方便對照閱讀。
