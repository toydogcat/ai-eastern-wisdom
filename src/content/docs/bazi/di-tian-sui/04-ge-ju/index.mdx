# 形象格局：導讀

如何判斷一個八字是「貴命」、「富命」還是「平凡命」？在《滴天髓》中，這不再是簡單地查神煞或看十神名稱，而是要深入觀察五行氣息的**「形象」**與**「格局」**。

## 核心哲學：以氣為主

傳統命理常執著於「八格」（正官、偏財等），但《滴天髓》提醒我們：
> **「格格推詳，以殺為重；察其衰旺，究其順悖。」**

這意味著：
1. **先看日主強弱**：日主有沒有力量承擔財官？
2. **看五行氣勢**：這個八字整體的氣是清純的還是混濁的？是流通的還是阻塞的？
3. **看格局真假**：如果是從格或化格，是真從還是假從？

## 本篇架構

本章將帶領你進入《滴天髓》高階的判斷邏輯：

### 1. 形象論
研究「兩氣合而成象」與「五氣聚而成形」，學習如何從八字整體的「形狀」判斷層次。

### 2. 方局論
探討地支的三會方（寅卯辰）與三合局（亥卯未）如何影響力量的爆發。

### 3. 官殺混雜與從化
破解「官殺混雜必凶」的迷思，並學習如何分辨真假從化格。
