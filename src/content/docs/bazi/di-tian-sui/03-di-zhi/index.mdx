# 十二地支：總論

在《滴天髓》中，地支的地位與天干同等重要，但論述的角度更偏重於**「動靜」**與**「力量的變化」**。

## 核心觀念

地支代表了命運的**「根基」**與**「實力」**。如果天干是果實，地支就是樹根。

### 1. 動靜之理
*   **陽支（子、寅、辰、午、申、戌）**：性動且強。它們的反應非常迅速，吉凶應驗得很快。
*   **陰支（丑、卯、巳、未、酉、亥）**：性靜且專。它們的力量比較沈穩、專一，吉凶的顯現通常需要較長的時間。

### 2. 生、敗、庫的區別
*   **四長生（寅、申、巳、亥）**：這是五行氣息萌芽的地方，最怕被衝動。
*   **四旺/敗地（子、午、卯、酉）**：這是五行最專純的地方，逢沖時需要精確計算氣勢強弱。
*   **四墓庫（辰、戌、丑、未）**：這是五行收斂的地方，通常喜歡被沖開，讓裡面的寶藏（藏干）能被運用。

### 3. 刑沖穿害的優劣
《滴天髓》強調：**「支神只以沖為重，刑與穿兮動不動。」**
這意味著在所有的地支互動中，**「沖」**的力量最直接、最劇烈，而刑、害等關係則相對次要，需要視情況而定。

## 學習建議

學習地支篇，你必須建立起「力量動態圖」：
- 想像地支之間在互相碰撞、結合、或是沈睡。
- 重點觀察：誰沖了誰？誰合了誰？這場戰鬥誰贏了？
