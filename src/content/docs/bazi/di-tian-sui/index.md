# 滴天髓：八字命理之聖經

《滴天髓》被譽為中國命理學的「聖經」，是一部探討八字五行生剋制化、理氣源流的巔峰之作。相傳為宋代京圖撰、明代劉伯溫註，後經清代任鐵樵加入大量命例進行詳盡註解（即《滴天髓闡微》），使其成為當今命理研究者必讀的最高典籍。

## 為什麼要學《滴天髓》？

1. **核心理氣觀**：不同於一般偏重「神煞」的論命法，《滴天髓》強調五行之氣的源流、清濁、中和。
2. **深刻的生命觀**：將自然界的運轉（天道、地道）與人的命運（人道）合而為一。
3. **任鐵樵的命例實戰**：任氏以自身艱辛的命運與無數實例，破除了當時對「吉神」的迷信，揭示了真正的五行強弱判斷法。

## 學習地圖

本站將根據《滴天髓闡微》的體系，將內容分為以下幾個主要篇章：

### 一、 通神論
探討命理的哲學根基與總綱，包括天道、地道、人道，以及理氣、配合的關鍵。

### 二、 干支論
深入探討十天干、十二地支的性情、生剋與特殊變化。

### 三、 形象格局論
分析八字的整體氣勢、格局取法（官殺、從化、形象等）。

### 四、 源流清濁論
討論五行氣息的起點與終點，以及命局貴賤的根本來源。

### 五、 人生專論
具體分析富貴貧賤、吉凶壽夭、六親、性情、疾病等具體人生範疇。
