# 第一板塊：基礎理論

在《子平真詮》的命理體系中，**天干地支**不是死板的字符，而是宇宙五行之氣的具體載體。如果我們無法精確掌握乾支在時間、空間、動靜狀態下的交互規律，就無法真正判斷格局的成敗。

本板塊將帶領您深入探索陰陽五行的源流，剖析天干地支的性格與生肖背後的生剋本質。

---

## 📖 核心章節指南

### [第一章・十干十二支與五行真諦](./01-shi-gan-di-zhi)
探索十天干與十二地支的由來，理解五行作為「氣的流動」本質，以及乾支如何構建出時間與空間的二維坐標。

### [第二章・陰陽生剋與生死動靜](./02-yin-yang-sheng-ke)
深入探討「生與克」的真正含義，破解陰陽五行在「長生十二神」中的生死運轉軌跡，以及陽干與陰乾在動靜上的根本區別。

### [第三章・十干性情與合而不合](./03-tian-gan-he-hua)
剖析天干五合（甲己合、乙庚合等）的性情本質，解答在何種特殊命局下天干「合而不合」的微妙變化。

### [第四章・刑沖會合解法與旺衰真機](./04-xing-chong-hui-he)
掌握地支最複雜的「刑、沖、會、合」機制，理解如何通過合來解沖、以沖來解合，並破除「得時不旺、失時不弱」的傳統盲區。
