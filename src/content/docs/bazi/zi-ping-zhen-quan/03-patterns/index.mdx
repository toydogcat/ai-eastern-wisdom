# 第三板塊：格局取法與運勢

有了前兩個板塊的乾支基礎與用神邏輯，我們終於來到了《子平真詮》的最高實戰殿堂——**「八格取運」**。

沈孝瞻大師在這裡將命局精細劃分為不同的格局，為每一個格局配備了明確的**「成格條件」**、**「破格忌諱」**與**「歲運喜忌」**。掌握了這個板塊，您就掌握了為真實命盤進行大運流年批斷的黃金鑰匙。

---

## 📖 核心章節指南

### [第九章・官殺格局之法與取運](./01-guan-sha)
探討代表社會地位、名利與權力的「正官格」與「七殺格」。學習如何通過財印、食神制殺等手段，將官殺的威力化為己用，以及它們在行運中的喜忌轉化。

### [第十章・財印格局之法與取運](./02-cai-yin)
剖析代表世俗財富的「財格」與代表智慧、貴人、精神世界的「印綬格」。理解財印不相礙的深奧玄機與大運流年的推斷要領。

### [第十一章・食傷格局之法與取運](./03-shi-shang)
解密代表才華、智慧、口才與自我表達的「食神格」與「傷官格」。掌握食神生財、傷官配印在實戰大運中的具體批導法則。

### [第十二章・祿刃與雜格之法與取運](./04-lu-ren)
研究極強、極剛的自我型格局——「陽刃格」與「建祿月劫格」，以及其餘奇特格局的取用法度，學習強旺格局的獨特行運規則。
