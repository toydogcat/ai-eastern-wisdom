卦象結構： 上澤（兌） ＋ 下水（坎）

符號概念： 沼澤底下破了洞，水全漏光了。上面是乾涸的湖底，下面是深陷的危險。

### 📖 白話大意

<div class="original-text">
【困乏與受限】
無止盡的攀升（升）最終會觸及天花板，導致資源耗盡。「困」就是困頓、窮困、被限制住。這是《易經》四大難卦之一，代表你陷入了無路可走、彈盡援絕的窘境，外部的支援完全斷絕。


</div>

:::tip[🚀 給現代人的啟示]
 遭遇死鎖，堅守核心邏輯。這就像是系統遇到了 OOM（Out of Memory）錯誤或 API 請求被限流（Rate Limit）。外在環境已經把你逼到死角，此時向外求援是無效的。你必須安靜下來，在極其受限的條件下，確保核心進程（致命部位）不崩潰，等待系統資源釋放。這也是考驗一個工程師或領導者底氣的時候。
:::
