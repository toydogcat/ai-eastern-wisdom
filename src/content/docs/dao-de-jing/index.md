# 道德經

《道德經》，又名《老子》，相傳為春秋末期的思想家**老子**（約西元前571年－471年）所著。
全書共**八十一章**，約五千餘字，是道家哲學的核心經典。

## 版本說明

本站採用**王弼本**（王弼注本）為底本，此為目前流傳最廣、最為學界通行的版本。

| 版本 | 特點 |
|------|------|
| 王弼本（流傳版） | 魏晉玄學家王弼（226-249）注解，章節清晰，是後世研究的主要依據 |
| 河上公本 | 以養生修身詮釋，民間廣泛流傳 |
| 馬王堆帛書（甲/乙本） | 西漢出土，保留更古老字形，本站另闢專區收錄 |

## 篇章結構

《道德經》分為兩大部分：

- **道經**（第 1–37 章）：論述「道」的本質、天地萬物的運化
- **德經**（第 38–81 章）：論述「德」的實踐、修身治國之道

> ⚠️ 注意：帛書版的排列恰好相反，以「德」在前、「道」在後，故稱《德道經》。

## 核心概念

### 道（Tāo）
「道」是萬物的根本，無形無名，卻無所不在。它既是宇宙的本原，也是事物運行的規律。

### 德（Dé）
「德」是「道」在具體事物中的體現，是道的顯現與落實。

### 無為（Wúwéi）
不強為、不妄為，順應自然規律，以柔克剛，以靜制動。
