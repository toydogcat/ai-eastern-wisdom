# ai-eastern-wisdom
The [eastern wisdom](https://toydogcat.github.io/ai-eastern-wisdom/).

## PWA 安裝說明 📲

本專案已支援 **PWA（Progressive Web App）** 功能，您可以將本站安裝至手機、平板或電腦桌面，以獲得更流暢的離線瀏覽與啟動體驗。

### 💻 在電腦端安裝 (Chrome / Edge / Brave)
1. 使用 Chrome、Edge 或 Brave 瀏覽器打開 [東方智慧｜道法自然](https://toydogcat.github.io/ai-eastern-wisdom/)。
2. 觀察瀏覽器網址列右側，會出現一個 **「安裝」** 圖示（或是點擊右上角 `...` 選單）。
3. 選擇 **「安裝」**，即可將本站轉為桌面應用程式。

### 📱 在行動裝置安裝
#### iOS (Safari)
1. 在 iPhone 或 iPad 上使用 **Safari 瀏覽器** 開啟本站。
2. 點擊 Safari 底部工具列的 **「分享」** 按鈕（向上箭頭）。
3. 向下捲動並選擇 **「加入主畫面」**。
4. 點擊右上角的 **「新增」** 即可。

#### Android (Chrome)
1. 在 Android 裝置上使用 **Chrome 瀏覽器** 開啟本站。
2. 點擊網址列右側的 **「更多」** 圖示 (`...`)。
3. 選擇 **「加到主畫面」**（或「安裝應用程式」）。
4. 點擊 **「新增」** 即可完成。
